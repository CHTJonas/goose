Welcome

This is a demonstration web login service (WLS), using:

  • the Flask microframework for Python webapps
  • the ucam-wls library to implement the WLS side of the Ucam-Webauth protocol.

You can try out the functionality of the WLS using this link. The usernames are similar to those used by the Raven demo service at demo.raven.cam.ac.uk, and the publicly-disclosed signing key, with key ID 901, is used to sign responses from this WLS.

An expanded set of features can be tested from this test page.
